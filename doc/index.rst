=============
jwt_helper
=============

jwt_helper is a package designed to help with JWT token validation.

Why was jwt_helper built?
---------------------------
I needed a unified way to validate JWT Access-Tokens across other Projects so i created this helper library. My goal was to provide a simple way to
validate JWTs issued by different issuers in a simple and easy manner. With this Library the configuration in other Services using this should be straight forward and easy to use.
